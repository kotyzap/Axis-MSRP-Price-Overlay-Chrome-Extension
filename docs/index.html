01 — WHAT IT DOES

Four content scripts,
one shared price catalog

Everything updates live — no page reload when you flip currency or touch the chipset filter.

FEATURE / 01

Inline MSRP + chipset badges

Every recognized product tile gets a yellow price badge — hover for the Axis part number — plus a chipset label where CamStreamer has data, with a checkmark when that chipset runs CamStreamer ACAPs.

FEATURE / 02

Currency + lookup from the toolbar

One click switches $ USD / € EUR everywhere at once — or search the full catalog by model name or part number, on any page.

FEATURE / 03

Multi-select chipset filter

ARTPEC-9 / -8 / -6-7 / older / Ambarella, all at once — unlike the site's single-select dropdown. One click on CamStreamer ACAPs selects every chipset family the apps actually run on.

FEATURE / 04

Badges on axis.com search too

Search for a model (M1137) or a whole series (Q35) — titles get an exact price, a variant range, or a "from $X" badge.

02 — SORTING

Sort series by cheapest price

An optional toggle re-ranks every series so the one with the cheapest currently-visible product floats to the top — recalculated live as you filter by chipset.

03 — INSTALL

Load it unpacked

Not published to the Chrome Web Store, so it loads like any local dev build. Grab a packaged copy from the latest release, or clone the source repo directly.

  1. 1Download and unzip the latest release.
  2. 2Open chrome://extensions in Chrome.
  3. 3Turn on Developer mode (top-right toggle).
  4. 4Click Load unpacked and select the axis-msrp-extension folder.
  5. 5Open the Product Selector — badges and the chipset filter appear automatically.
  6. 6Click the extension icon any time to flip currency or look up a model by name or part number.
04 — DATA SOURCES

Where the numbers come from

PRICES / USD

catalog-data.js bundles 290 models / 377 SKUs extracted from Axis's Q1 2026 "cheat sheet" price list PDF (MSRP, USD). Coverage isn't 100% — a handful of very new SKUs, VMS licenses, and some accessories aren't priced individually there.

PRICES / EUR

The same file carries an msrp_eur figure per SKU, joined from the Axis EE January 2026 price list. That source only covers camera products — encoders, recorders, audio, access control, 2N, and accessories have no EUR data. No badge shows in EUR mode rather than guessing.

CHIPSETS

chipset-data.js bundles a snapshot parsed from camstreamer.com's supported-hardware list — refreshed weekly by the background service worker, or on demand from the popup.
