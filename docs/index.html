01 — WHAT IT DOES

Every axis.com surface,
one shared price catalog

Product Selector, search, product/category pages, and Compare Products tables — everything updates live, no page reload when you flip currency, refresh chipset data, or touch the chipset filter.

FEATURE / 01

Inline MSRP + chipset badges

Every recognized product tile gets a yellow price badge — hover for the Axis part number — plus a chipset label where CamStreamer has data, with a checkmark for CamStreamer Support.

FEATURE / 02

Currency + lookup from the toolbar

One click switches € EUR / $ USD everywhere at once (EUR is the default) — or search the full catalog by model name or part number, on any page. Toggle light/dark theme from the same header.

FEATURE / 03

Multi-select chipset filter

ARTPEC-9 / -8 / -6-7 / older / Ambarella, all at once — unlike the site's single-select dropdown. One click on CamStreamer Support selects every chipset family the apps actually run on.

FEATURE / 04

Badges on axis.com search too

Search for a model (M1137) or a whole series (Q35) — titles get an exact price, a variant range, or a "from €X" badge.

FEATURE / 05

Product and category pages, badged too

Every model tile on a hub page (Box cameras) or series page gets a top-of-card price + chipset badge — a specific price for one model, or a "from €X" range for a whole series card — without ever covering the site's own captions.

FEATURE / 06

Series price range in the breadcrumb, plus Compare Products rows

A series page's breadcrumb trail gets the full price range and every chipset used across the series in one glance. Open the Compare Products table on the same page and it gains two new rows at the top — Chipset and Price — one column per model.

07 — SORTING

Sort series by cheapest price

An optional toggle re-ranks every series so the one with the cheapest currently-visible product floats to the top — recalculated live as you filter by chipset. TRY IT — CHECK THE BOX

OFF Series stay in the Product Selector's own default order (M11 → M30 …). ON Modular cameras jumps ahead of M30 — now the cheapest visible series.

08 — MONTHLY UPDATES

Drop in any AXIS Price List — no reinstall

Axis publishes a new AXIS Price List roughly every month. Since v2.0.0, the extension parses it itself: open the settings page from the toolbar popup's ⚙ button and drag the new file in.

  • Known format, recognized automatically. Drop the usual monthly .xls/.xlsx file and it reads the "All products" sheet, detects the name/part-number/price columns, and figures out EUR vs. USD on its own.
  • Anything else, mapped by hand. A different layout or a one-off export brings up a column-mapping panel with a live preview, so you can point out the sheet, header row, and price/part-number columns yourself.
  • Coverage report before you commit. See how many SKUs matched by part number vs. name fallback, a breakdown by product section, and anything unmatched — then Apply update, or Revert to bundled prices at any time.
  • Runs entirely in your browser. Parsing happens locally via a bundled copy of SheetJS — the file itself never leaves your machine. The only network call is an EUR/USD reference-rate lookup (ECB data via api.frankfurter.dev, no API key).
09 — INSTALL

Install from the Chrome Web Store

No developer mode, no unpacked folder — install it like any other extension straight from the Chrome Web Store listing. Prefer to build from source instead? The GitHub repo is still there.

  1. 1Open the Axis MSRP Price Overlay listing on the Chrome Web Store.
  2. 2Click Add to Chrome, then confirm in the permissions prompt.
  3. 3Open the Product Selector — badges and the chipset filter appear automatically.
  4. 4Click the extension icon any time to flip currency, look up a model, or open settings to refresh the monthly price list.
  5. 5Updates are pushed automatically through the Web Store — no manual reinstall needed for future releases.
10 — DATA SOURCES

Where the numbers come from

PRICES / EUR

The "All products" sheet of the AXIS Price List — 290 models, 377 SKUs, bundled with Q2 2026 pricing by default and refreshable any month by dropping in a new file.

PRICES / USD

FX-derived from the EUR figure at the prevailing EUR/USD rate. SKUs without EUR data fall back to Axis's Q2 2026 US price list PDF.

CHIPSETS

chipset-data.js bundles a snapshot parsed from camstreamer.com's supported-hardware list — refreshed weekly by the background service worker, or on demand from the popup.
