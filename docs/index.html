What it does

Four content scripts, one shared price catalog. Everything updates live — no page reload needed when you flip currency or touch the chipset filter.

1

Inline MSRP + chipset badges

Every recognized product tile on the Product Selector gets a yellow price badge (hover for the Axis part number) and, where CamStreamer has data for it, a small chipset label above it — with a checkmark when that chipset runs CamStreamer ACAPs.

2

Currency + model lookup, from the toolbar

Click the extension icon to switch $ USD / € EUR everywhere at once, or search the full catalog directly by model name or part number — independent of whichever Axis page you're on.

3

Chipset filter panel

Multi-select ARTPEC-9 / ARTPEC-8 / ARTPEC-6-7 / older ARTPEC / Ambarella — unlike the site's own single-select "System-on-chip" dropdown. One click on CamStreamer ACAPs selects every chipset family CamStreamer apps actually run on.

4

Price badges on axis.com search too

Search axis.com/search for a model (M1137) or a whole series (M11, Q35) — recognized titles get an exact price, a variant range, or a "from $X" badge for series spanning multiple models.

Sort series by cheapest price

An optional toggle in the chipset filter panel re-ranks every series so the one with the cheapest currently-visible product floats to the top — recalculated live as you filter by chipset.

Install (unpacked)

Not published to the Chrome Web Store, so it loads as an unpacked extension — same as any local dev build.

  1. Open chrome://extensions in Chrome.
  2. Turn on Developer mode (top-right toggle).
  3. Click Load unpacked and select the axis-msrp-extension folder.
  4. Open the Product Selector — badges and the chipset filter panel appear automatically.
  5. Click the extension icon any time to flip currency or look up a model by name or part number.

Data sources

Prices (USD)

catalog-data.js bundles 290 models / 377 SKUs extracted from Axis's Q1 2026 "cheat sheet" price list PDF (MSRP, USD). Coverage isn't 100% — a handful of very new SKUs, VMS software licenses, and some accessory items aren't priced individually there.

Prices (EUR)

The same file carries an msrp_eur figure per SKU, joined from the Axis EE January 2026 price list. That source only covers actual camera products — encoders, recorders, network audio, access control, 2N, and accessories simply have no EUR data. No badge shows in EUR mode rather than guessing.

Chipsets

chipset-data.js bundles a snapshot parsed from camstreamer.com's supported-hardware list — inferred from which CamStreamer app build each device runs, refreshed weekly by the background service worker (or on demand from the popup).
